TextLink

This component renders a native `a` element by default, but this can be customised via the `linkComponent` prop on BraidProvider.
If you want to render something that looks like a link but isn’t semantically a link (e.g. a button), you should use TextLinkRenderer instead.
Please note that this component must be nested within a Text, Heading or Actions component.

Examples

Text Link

Text link
<Text>
  <TextLink hitArea="large" href="#">
    Text link
  </TextLink>
</Text>
